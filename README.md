# openshorts.dev
openshorts to baza darmowych narzędzi, bibliotek i zasobów UI dla deweloperów. Projekt powstał, aby ułatwić znajdowanie sprawdzonych assetów bez przedzierania się przez dziesiątki zakładek w przeglądarce.

## Krótko o projekcie
Projekt miał być swojego typu testem możliwości dla AstroJS, gdy zaczynałem się go uczyć. Nadal potrzebuje on usprawnień, na razie został wstrzymany w dalszym rozwoju ale działa jako aktywna i aktualna biblioteka różnego rodzaju zasobów dla innych.

## Główne funkcje:
    - Możliwość dodawania swoich własnych lub znalezionych przez użytkowników stron przydatnych dla innych, wraz z tytułe, opisem i otagowaniem.
    - Wyszukiwarka AI, wyszukiwanie wektorowe na podstawie tytułu, opisu oraz tagów strony, która interesuje użytkownika na podstawie prompta.
    - Prosty blog (niestety bez systemu CMS ze względu na funkcje czysto testową pod próby polepszenia SEO)
    - Możliwość komentowania stron oraz "polubiania" by zapisać je jako ulubione.
    - System profilów

## Projekt został zbudowany przy użyciu:
    - AstroJS
    - ReactJS
    - Typescript
    - PostgresSQL (NeonDB)
    - TailwindCSS
