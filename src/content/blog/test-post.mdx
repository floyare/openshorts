## Introduction

Astro is a modern static site builder that allows you to build fast websites using your favorite frameworks.

## Why Use Astro?

- **Performance:** Ships less JavaScript by default.
- **Flexibility:** Use React, Vue, Svelte, and more.
- **Simplicity:** Easy to get started.

## Getting Started

1. Install Astro:
     ```bash
     npm create astro@latest
     ```
2. Follow the prompts to set up your project.
3. Start the development server:
     ```bash
     npm run dev
     ```
